# Privacy Policy — greenit plugin

_Last updated: 2026-07-08_

This privacy policy covers the **`greenit`** plugin for Claude Code (the "Plugin"), distributed at
https://github.com/drnoa/greenit_agentic_plugin.

## Summary

The Plugin collects **no personal data**. It contains no telemetry, analytics, tracking, or
advertising, and it transmits nothing to the author or to any third party.

## What the Plugin is

The Plugin consists solely of static Markdown files — skills (`skills/<name>/SKILL.md`) and one
read-only reviewer agent (`agents/green-reviewer.md`). These are instructions that run inside your
local Claude Code session. The Plugin ships no executable code, no hooks, no MCP servers, and no
background processes.

## Data collection

- The Plugin **does not collect, store, or process** any personal information.
- The Plugin **does not create accounts, cookies, identifiers, or logs**.
- The author **receives no data** of any kind from your use of the Plugin.

## Data transmission and network activity

- The Plugin makes **no network requests on its own.**
- Skill content references external documentation URLs (e.g. Green Software Foundation, Electricity
  Maps, Boavizta, web.dev). These are informational links only; nothing is contacted unless **you**
  choose to open a link or ask Claude to fetch one. Any such request is governed by that third
  party's own privacy policy.

## Your code and prompts

The Plugin runs within Claude Code and operates on the files and prompts of your active session. That
processing is performed by Claude Code / the Anthropic API, and is governed by Anthropic's privacy
policy (https://www.anthropic.com/legal/privacy), not by the Plugin. The Plugin itself neither
retains nor exports your code, prompts, or results.

## Children's privacy

The Plugin is a developer tool, is not directed at children, and collects no data from anyone.

## Changes to this policy

Updates will be published to this file in the repository, with a revised "Last updated" date above.
