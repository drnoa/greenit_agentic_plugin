Claude Code plugin

greenit

Make Green IT patterns the default, low-friction path in software development — energy-, carbon-, and resource-efficient design, implementation, and review.

View on GitHub Install

What it does

Encodes the 7 technical Green IT principles plus Software Carbon Intensity measurement and impact-focus as model-invoked skills. Claude applies the right pattern automatically when your task matches — and a read-only agent audits a diff or project against all of them.

Skills

calculate-sci

Software Carbon Intensity SCI = (E×I+M)/R (ISO/IEC 21031:2024) for your project.

find-green-levers

Find the highest-impact levers before optimizing (Pareto).

minimize-footprint

Less is more — don't build, store, or run what isn't needed.

reduce-data-transfer

Network efficiency — cut transferred and stored bytes.

efficient-compute

Algorithmic efficiency — minimize and avoid computation.

optimize-resource-usage

Hardware efficiency — maximize utilization, kill idle draw.

carbon-aware-scheduling

Carbon awareness — shift workloads to greener regions/times.

hardware-aware-optimization

Exploit hardware: OLED, ARM/Graviton, crypto units, PUE.

extend-longevity

Extend device & software lifetime to cut embodied carbon.

green-reviewer agent

Read-only audit against the 7 principles; impact-ranked findings routed to a fix skill.

Example use cases

  • Estimate a service's carbon footprint — “What's the SCI of this API?” → gCO₂eq per 1000 requests with a confidence level.
  • Decide where to start — “Where should I optimize this project for energy?” → biggest structural levers, ranked by impact.
  • The right pattern while coding, automatically — editing images surfaces network efficiency; adding a cron job surfaces carbon-aware scheduling; an ORM query flags N+1.
  • Review a change — “Run a Green IT audit on my diff” → path:line — principle — problem — fix.
  • Cut mobile battery drain — dark mode / animations surface true-black OLED and transform/opacity guidance.
  • Lower infra emissions without code changes — region choice, ARM instances, right-sizing, scale-to-zero.

Install

/plugin marketplace add anthropics/claude-plugins-community
/plugin install greenit@claude-community

Try it locally without installing:

git clone https://github.com/drnoa/greenit_agentic_plugin.git
claude --plugin-dir ./greenit_agentic_plugin
